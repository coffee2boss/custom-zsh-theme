# custom-zsh-theme
custom zsh theme using powerlevel10k and ohmyzsh

requires:
1. zsh as default shell
2. ohmyzsh                                                                         - https://github.com/ohmyzsh/ohmyzsh
3. powerlevel10k                                                                   - https://github.com/romkatv/powerlevel10k
4. Icons require mesoloLGS Nerd Font intalled and used by ( wsl or linux distro )  - https://www.nerdfonts.com/font-downloads



Contains:
.ohmyzsh ( with powerlevel10k installed) - Add to home dir
.zshrc lines for custom-theme ( add to your .zshrc )
.p10 theme - Add to home dir alongside .zshrc
